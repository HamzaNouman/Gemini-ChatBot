## How to make it work

1. Clone the repository
```bash
git clone https://github.com/HamzaNouman/Gemini-ChatBot
```

2. Create a venv
```bash
python -m venv venv
```

3. Install the requirments
```bash
pip install -r requirements.txt
```

4. Create a .env file and add your gemini api key
```bash
.env=YOUR_API_KEY_HERE
```
